API de Cadastro

Este é um projeto de estudo desenvolvido em PHP com MySQL, que tem como objetivo criar uma API simples para cadastrar e consultar pessoas.

Ele inclui:

Arquivo principal da API (api.php)

Arquivo de conexão com o banco (conexao.php)

Script SQL para criação do banco e da tabela (BD api_cadastro.sql)

O projeto foi feito para praticar conceitos de desenvolvimento backend em PHP, manipulação de banco de dados e versionamento com Git/GitHub.
